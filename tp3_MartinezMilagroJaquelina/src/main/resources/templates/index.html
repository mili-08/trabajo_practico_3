Programacion Visual - APU-2008

La materia Programación Visual es un curso integral diseñado para proporcionar a los estudiantes un sólido entendimiento de los principios fundamentales y las tecnologías modernas utilizadas en el desarrollo de aplicaciones web con Java. Desde sus bases en el lenguaje Java hasta el desarrollo de aplicaciones robustas utilizando Spring Boot y herramientas asociadas, este curso está diseñado para equipar a los participantes con las habilidades necesarias para enfrentar los desafíos del desarrollo web en el mundo contemporáneo.

¿Qué vas a aprender?

  • Enfoque Integral
  • Fundamentos de Java
  • Programación Orientada a Objetos (POO)
  • Introducción a Spring Boot
  • Gestión de Dependencias y Versionamiento
  • Desarrollo de Aplicaciones Web
  • Persistencia de Objetos con JPA
  • Desarrollo Practico

Requerimientos académicos

  1. Conocimientos Básicos de Programación: Es recomendable tener conocimientos básicos de programación, incluyendo conceptos como variables, tipos de datos, estructuras de control y funciones.
  2. Alfabetización digital: Conocimientos relacionados a la descarga e instalación de programas
  3. Habilidades de resolución de problemas: Capacidad de abordar problemas complejos de manera sencilla.
